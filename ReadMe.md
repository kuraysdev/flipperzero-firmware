# Цонтрибутинг

Оур маин гоал ис то буилд а хеалтхю анд сустаинабле цоммунитю ароунд Флиппер, со ве'ре опен то аню нев идеас анд цонтрибутионс. Ве алсо хаве соме рулес анд табоос хере, со плеасе реад тхис паге анд оур [Цоде Оф Цондуцт](/CODE_OF_CONDUCT.md) царефуллю.

## Ай нид хелп

Тхе бест плейс то сёрч фор ансверс ис оур [Усер Доцументатион](https://docs.flipper.net). Иф юу кант файнд тхе ансвер тхере, чецк оур [Дисцорд Сервер](https://flipp.dev/discord) ор оур [Форум](https://forum.flipperzero.one/). Иф юу вант то цонтрибуте то тхе фирмваре девелопмент ор модифи ит фор ёур оун неедс, юу кан алсо чецк оур [Девелопер Доцументатион](https://developer.flipper.net/flipperzero/doxygen).

## Ай вант то репорт ан иссуе

Иф ёу'ве фоунд ан иссуе анд вант то репорт ит, плеасе чецк оур [Иссуес](https://github.com/flipperdevices/flipperzero-firmware/issues) пейдж. Маке шуре тхе десцриптион цонтэйнс информатион абаут тхе фирмваре версион ёу'ре усинг, ёур платформ, анд а клеар експланацион оф тхе степс то репродюце тхе иссуе.

## Ай вант то цонтрибуте коде

Бефоре опенинг а ПР, плеасе цонфирм тхат ёур ченджес муст бе цонтэйнед ин тхе фирмваре. Мэни идеас кан еасили бе имплементед ас екстернал аппликационс анд публикед ин тхе [Флиппер Аппликацион Цаталог](https://github.com/flipperdevices/flipper-application-catalog). Иф ёу ар уншуре, реач аут то ус он тхе [Дисцорд Сервер](https://flipp.dev/discord) ор тхе [Иссуес](https://github.com/flipperdevices/flipperzero-firmware/issues) пейдж, анд ве'лл хелп ёу файнд тхе ригхт плейс фор ёур коде.

Алсо, плеасе реад оур [Цонтрибутион Гуиде](/CONTRIBUTING.md) анд оур [Кодинг Стиле](/CODING_STYLE.md), анд маке шуре ёур коде ис цомпатибле витх оур [Прожецт Лиценсе](/LICENSE).

Финалли, опен а [Пулл Рекуест](https://github.com/flipperdevices/flipperzero-firmware/pulls) анд маке шуре тхат ЦИ/ЦД статусес ар алл греен.

# Девелопмент

Флиппер Зеро Фирмваре ис вриттен ин Ц, витх соме битс анд писес вриттен ин Ц++ анд армв7м ассембли ленгуагес. Ан интермедиате левел оф Ц ноуледж ис рекомендет фор цомфортэбле програмиинг. Ц, Ц++, анд армв7м ассембли ленгуагес ар суппортед фор Флиппер аппликационс.

# Фирмваре РоадМап

## Рекуирементс

Суппортед девелопмент платформс:

- Виндоус 10+ витх ПоверШелл анд Гит (х86_64)
- макОС 12+ витх Цомманд Лине тоолс (х86_64, арм64)
- Убунту 20.04+ витх билд-ессентиал анд Гит (х86_64)

Суппортед ин-цирцит дебуггерс (оптионал бут хигхли рекомендет):

- [Флиппер Зеро Ви-Фи Девелопмент Боард](https://shop.flipperzero.one/products/wifi-devboard)
- ЦМСИС-ДАП компатибле: Распберри Пи Дебуг Пробе анд етц...
- СТ-Линк (в2, в3, в3модс)
- Ж-Линк

Флиппер Билд Систем вилл таке царе оф алл тхе отхер депенденцис.

## Цлонинг соурце коде

Маке шуре ёу хаве еноугх спаце анд цлоне тхе соурце коде:

```shell
git clone --recursive https://github.com/flipperdevices/flipperzero-firmware.git
```

## Билдинг

Билд фирмваре усинг Флиппер Билд Тоол:

```shell
./fbt
```

## Флашинг фирмваре усинг ан ин-цирцит дебуггер

Цоннецт ёур ин-цирцит дебуггер то ёур Флиппер анд флаш фирмваре усинг Флиппер Билд Тоол:

```shell
./fbt flash
```

## Флашинг фирмваре усинг УСБ

Маке шуре ёур Флиппер ис он, анд ёур фирмваре ис фунцтионинг. Цоннецт ёур Флиппер витх а УСБ цабле анд флаш фирмваре усинг Флиппер Билд Тоол:

```shell
./fbt flash_usb
```

# Пројецт структуре

- `applications`        - Аппликационс анд сервисес усед ин фирмваре
- `applications_users`  - Плаце фор ёур аддитионал аппликационс анд сервисес
- `assets`              - Ассетс усед би аппликационс анд сервисес
- `documentation`       - Доцументатион генератион систем цонфигс анд инпут филес
- `furi`                - Фури Цоре: ОС-левел примитивес анд хелперс
- `lib`                 - Оур анд 3рд парти либрариес, драйверс, тоолс анд етц...
- `site_scons`          - Билд систем цонфигуратион анд модулес
- `scripts`             - Супплементари скриптс анд вариоус питхон либрариес
- `targets`             - Фирмваре таргетс: платформ специфик коде

Алсо, сее `ReadMe.md` филес инсиде тхосе директориес фор фуртхер деталс.
