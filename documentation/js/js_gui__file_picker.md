# АПИ референце

## пицкФиле()
Дисплайс а филе пицкер анд ретюрнс тхе селецтед филе, ор юндефинед иф цанцеллед.

**Параметерс**
  - `basePath`: тхе патх то старт ат
  - `extension`: тхе филе ехтенсион(с) то схош (лике `.sub`, `.iso|.img`, `*`)

**Ретюрнс**

А `string` патх, ор `undefined`.
