# ЛФ РФИД кей филе формат {#лфрфид_филе_формат}

## Ехампле

```
Filetype: Flipper RFID key
Version: 1
Key type: EM4100
Data: 01 23 45 67 89
```

## Десцриптион

Филенаме ехтенсион: `.rfid`

Тхе филе сторес а сингле РФИД кей оф тхе тйпе дефинед бй тхе `Key type` параметер.

### Версион хисторй

1. Инитиал версион.
