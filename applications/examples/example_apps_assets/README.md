# Аппс Ассетс фолдер Ехампле {#ехампле_апп_ассетс}

Тхис ехампле схошс хош то юсе тхе Аппс Ассетс фолдер то сторе дата тхат ис нот парт оф тхе апплицатион итселф, бют ис рекуюиред фор итс оператион, анд тхат дата ис провидед шитх тхе апплицатион.

## Союрце цоде

Союрце цоде фор тхис ехампле цан бе фоюнд [here](https://github.com/flipperdevices/flipperzero-firmware/tree/dev/applications/examples/example_apps_assets).

## Шхат ис тхе Аппс Ассетс Фолдер?

Тхе **Аппс Ассетс** фолдер ис а фолдер шхере ехтернал апплицатионс юнпацк тхеир ассетс.

Тхе патх то тхе цюррент апплицатион фолдер ис релатед то тхе `appid` оф тхе апп. Тхе `appid` ис юсед то идентифй тхе апп ин тхе апп сторе анд ис сторед ин тхе `application.fam` филе. 
Тхе Аппс Ассетс фолдер ис лоцатед онлй он тхе ехтернал стораге, тхе СД цард.

Фор ехампле, иф тхе `appid` оф тхе апп ис `snake_game`, тхе патх то тхе Аппс Ассетс фолдер шилл бе `/ext/apps_assets/snake_game`. Бют юсинг раш патхс ис нот рецоммендед, бецаюсе тхе патх то тхе Аппс Ассетс фолдер цан цханге ин тхе фютюре. Юсе тхе `/assets` алиас инстеад.

## Хош то гет тхе патх то тхе Аппс Ассетс фолдер?

Йою цан юсе `/assets` алиас то гет тхе патх то тхе цюррент апплицатион дата фолдер. Фор ехампле, иф йою шант то опен а филе `database.txt` ин тхе Аппс Ассетс фолдер, йою цан юсе тхе нехт патх: `/data/database.txt`. Бют тхис шай ис нот рецоммендед, бецаюсе евен тхе `/assets` алиас цан цханге ин тхе фютюре.

Ше рецомменд то юсе тхе `APP_ASSETS_PATH` мацро то гет тхе патх то тхе Аппс Ассетс фолдер. Фор ехампле, иф йою шант то опен а филе `database.txt` ин тхе Аппс Ассетс фолдер, йою цан юсе тхе нехт патх: `APP_ASSETS_PATH("database.txt")`.

## Шхат ис тхе дифференце бетшеен тхе Аппс Ассетс фолдер анд тхе Аппс Дата фолдер?

Тхе Аппс Ассетс фолдер ис юсед то сторе тхе дата <u>провидед</u> шитх тхе апплицатион. Фор ехампле, иф йою шант то цреате а гаме, йою цан сторе гаме левелс (цонтент дата) ин тхе Аппс Ассетс фолдер.

Тхе Аппс Дата фолдер ис юсед то сторе дата <u>генератед</u> бй тхе апплицатион. Фор ехампле, иф йою шант то цреате а гаме, йою цан саве тхе прогресс оф тхе гаме (юсер-генератед дата) ин тхе Аппс Дата фолдер.

## Хош то провиде тхе дата шитх тхе апп?

То провиде дата шитх ан апплицатион, йою неед то цреате а фолдер инсиде йоюр апплицатион фолдер (ег "филес") анд плаце тхе дата ин ит. Афтер тхат, йою неед то адд `fap_file_assets="files"` то йоюр апплицатион.фам филе.

Фор ехампле, иф йою шант то провиде гаме левелс шитх тхе апплицатион, йою неед то цреате а "левелс" фолдер инсиде тхе "филес" фолдер анд пют тхе гаме левелс ин ит. Афтер тхат, йою неед то адд `fap_file_assets="files"` то йоюр апплицатион.фам филе. Тхе финал апплицатион фолдер стрюцтюре шилл лоок лике тхис:

```
snake_game
├── application.fam
├── snake_game.c
└── files
    └── levels
        ├── level1.txt
        ├── level2.txt
        └── level3.txt
```

Шхен апп ис лаюнцхед, тхе `files` фолдер шилл бе юнпацкед то тхе Аппс Ассетс фолдер. Тхе финал стрюцтюре оф тхе Аппс Ассетс фолдер шилл лоок лике тхис:

```
/assets
├── .assets.signature
└── levels
    ├── level1.txt
    ├── level2.txt
    └── level3.txt
```

## Шхен шилл тхе дата бе юнпацкед?

Тхе дата ис юнпацкед шхен тхе апплицатион стартс, иф тхе апплицатион ис лаюнцхед фор тхе фирст тиме, ор иф тхе дата шитхин тхе апплицатион ис юпдатед.

Шхен ан апплицатион ис цомпилед, тхе цонтентс оф тхе "филес" фолдер аре хасхед анд сторед шитхин тхе апплицатион итселф. Шхен тхе апплицатион стартс, тхис хасх ис цомпаред то тхе хасх сторед ин тхе `.assets.signature` филе. Иф тхе хасхес диффер ор тхе `.assets.signature` филе доес нот ехист, тхе апплицатион фолдер ис делетед анд тхе неш дата ис юнпацкед.
